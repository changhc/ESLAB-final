# ESLAB-project

## Prerequisite

* mjpg-streamer

* Raspberry Pi

* Webcam

* Motor (MG995)

* flask

* pigpio

## Usage

1. Set up your Raspberry Pi environment

2. Determine variables ```cmd``` and ```path``` ```in server.py```.

3. ```python server.py```

4. ```http://192.168.0.1:5000``` for website ```http://192.168.0.1:9090``` for mjpg streaming
